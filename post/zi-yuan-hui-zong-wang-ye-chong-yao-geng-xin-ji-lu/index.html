——2023-4-30 09:58

弹窗修复成功，还要感谢github@winteru
彈窗不能正確顯示的原因：
<body> 中的onload函數和window.onload實際上是一個函數，在 <body> 指定onload後，又在第38行中再次設置了onload，導致原來有的onload被裝蓋，無法觸發showPopup函數。

解決方法：
將window.onload換為window.addEventListener('load', f)，addEventListener不會掩蓋已經有的onload，而是會在原有基礎上添加。
若干HTML標籤沒有關閉的問題:
一些html標籤沒有正確關閉
建議將HTML代碼進行一下對齊，便於發現此類問題.可以用pret層級自動對齊代碼。
body元素在div裡面的問題:
按HTML的要求，body應該直接屬於html，不應該包含在其他元素裡面。

今天给li标签添加了相对定位，li里面的a添加了绝对定位。

li {
        background-color: #ffffff;
        border-radius: 10px;
        box-shadow: 0px 0px 10px #cccccc;
        margin: 20px;
        padding: 20px;
        text-align: center;
        width: 300px;
        cursor: pointer;
        animation: fadeIn 1s ease-in-out;
        position: relative;
   }

li a {
        font-family: 'Minecraft-Regular';
        position: absolute;
        top: 0;
        left: 0;
        width: 100%;
        height: 100%;
}

li:hover {
         transform: scale(1.05);
}

li a:hover {
        display: none;
}

li a:active {
        display: none;
}

效果就是，点击li就能触发a。
虽然这样直接修改a好像也能做到……

今天更新了二级导航，我看教程视频上基本需要使用定位来完成，我用了个简单的办法，以下是代码：

//html部分
        <div class="navbar">
                <ul class="navlist">
                    <li class="nav-link">
                        <a href="#anime" target="_self">动画</a>
                        <ul class="droplist">
                            <li>
                                <a href="#anime-legal" target="_self">正版</a>
                                <a href="#anime-apps" target="_self">软件</a>
                            </li>
                        </ul>
                    </li>
    
    //css部分
    .nav-link {
    float: left;
    width: 100px;
    line-height: 40px;
    padding-left: 50px;
    text-align: center;
    }

    .navlist a {
        color: rgba(1, 0, 0, 0.956);
        font-size: 20px;
        font-family: 'Default',
        'Microsoft YaHei',
        'tahoma',
        'arial';
        line-height: 20px;
        display: block;
        animation: none;
    }

    .nav-link:hover {
        background-color: #fff;
    }

    .navbar a {
        padding: 15px;
    }

    .navbar a:visited {
        display: none;
    }

    .navbar a:hover {
        background: none;
        border-radius: none;
        color: rgb(90, 92, 6);
    }

    .navbar ul {
        display: block;
        flex-wrap: none;
        justify-content: none;
        list-style: none;
        margin: 0;
        padding: 0;
        text-transform: none;
    }
